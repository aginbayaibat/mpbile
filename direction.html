  • Frontend

    Frontend — браузер оқи алатын, көрсете алатын және/немесе іске қоса алатын кез келген нәрсе. Яғни, бұл HTML, CSS және JavaScript.

    Стек:
    • HTML
    • CSS
    • JavaScript
    HTML (HyperText Markup Language) браузерге «тақырып», «абзац», «тізім», «тізім элементі» сияқты бет мазмұнының қандай екенін айтады.
    CSS (Cascading Style Sheets) браузерге элементтерді қалай көрсету керектігін айтады, мысалы, «бірінші абзацтан кейін 20 пиксель шегінісі» немесе «негізгі элементтегі барлық мәтін қою сұр және Verdana шрифтінде жазылуы керек».
    JavaScript шолғышқа жеңіл бағдарламалау тілін пайдаланып белгілі бір әрекеттесулерге қалай жауап беру керектігін айтады. Көптеген сайттар шынымен JavaScript-ті көп қолданбайды, бірақ егер сіз бірдеңені бассаңыз және беттің мазмұны ақ экран жыпылықтамай өзгерсе, JavaScript бір жерде қолданылған.
  • Bacend

    Аты айтып тұрғандай, Backend сайттың және сервердің ішкі бөлігі (ағыл. Back - back). Жалпы айтқанда, бұл сайт пен сервердің дұрыс жұмыс істеуіне мүмкіндік беретін бағдарламалық-аппараттық кешен.

    Стек:
    • Java
    • PHP
    • Python
    • MySQL
    • PostgreSQL
    Java Sun Microsystems (кейінірек Oracle сатып алған) әзірлеген қатты типтелген жалпы мақсаттағы объектіге бағытталған бағдарламалау тілі.
    PHP (Ағылшынша PHP: Hypertext Preprocessor - "PHP: гипермәтіндік препроцессор"; бастапқыда PHP / FI (Personal Home Page / Form Interpreter), кейінірек Personal Home Page Tools[11] - " Құралдар" деп аталды. «Жеке веб-беттерді жасау үшін») — бұл веб-қосымшаларды әзірлеу үшін кеңінен қолданылатын сценарий тілі.
    MySQL реляциялық мәліметтер базасын басқарудың ақысыз жүйесі. MySQL-ті Oracle корпорациясы әзірлейді және қолдайды, ол бұрын MySQL AB швед компаниясын сатып алған Sun Microsystems компаниясымен бірге сауда белгісі құқықтарын иеленді. Дәл осы тапсырыстың арқасында репликация механизмі дерлік ертерек нұсқаларда пайда болды.
  • Data Science

    Data Science үлкен деректермен жұмыс істейді. Үлкен деректер - бұл құрылымдалмаған ақпараттың үлкен көлемі: мысалы, белгілі бір кезеңдегі ауа-райы деректері, іздеу жүйелеріндегі сұрау статистикасы, спорттық ұпайлар, микробтық геномдар деректер базасы және т.б. Мұндағы негізгі сөздер «үлкен көлем» және «құрылымсыз». Мұндай деректермен жұмыс істеу үшін математикалық статистика және машиналық оқыту әдістері қолданылады.

    Стек:
    • Excel, Google кестелері
    • Python, NumPy и Pandas кітапханалары
    • SQL, Power BI
    Excel, Google кестелері Excel (кейде Microsoft Office Excel[5] деп те аталады) — Microsoft Windows, Windows NT және Mac OS, сондай-ақ Android, iOS және Windows Phone үшін Microsoft жасаған электрондық кесте бағдарламасы. Ол экономикалық және статистикалық есептеулерді, графикалық құралдарды және Mac OS X жүйесіндегі Excel 2008 нұсқасын қоспағанда, VBA (Visual Basic for Application) макробағдарламалау тілін қамтамасыз етеді. Microsoft Excel бағдарламасы Microsoft Office бағдарламасының бөлігі болып табылады.
    Python, NumPy и Pandas NumPy «Numeric Python» дегенді білдіреді, оны «Numeric Python» деп аударуға болады. Бұл кітапхана көп өлшемді массивтермен жұмыс істеу мүмкіндігін береді. Ол деректер түрі мен өлшемі бірдей элементтердің көп өлшемді контейнері ndarray класына негізделген. Бұл класстың объектілері жадты аз алады және олармен операциялар әдеттегі Python тізімдеріне қарағанда жылдамырақ орындалады.
    SQL Ағылшын тілінен SQL. Structured Query Language – «құрылымдық сұраныс тілі») – сәйкес дерекқорды басқару жүйесімен басқарылатын реляциялық деректер базасындағы деректерді құру, өзгерту және басқару үшін қолданылатын декларативті бағдарламалау тілі.
